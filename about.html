About NexScan GmbH

We are a Vienna-based security software company on a mission to make code security accessible to every development team — not just large enterprises.

Our Story

Built by developers,
for developers

NexScan was founded in 2023 by a team of software engineers frustrated by security tools that were either too expensive, too slow, or required a dedicated security team to operate.

We built DevScan to be the tool we always wanted — one that gives developers immediate, actionable feedback without leaving their workflow. Today, DevScan protects repositories across Europe and beyond.

Our company is headquartered in Vienna's 9th district. We believe security and developer experience are not a tradeoff — and we are proving it, one scan at a time.

Brand

Our brand marks

NexScan GmbH uses three distinct brand forms. These marks are protected and may not be used without written permission. We verified that none of our marks conflict with existing registrations via EUIPO eSearch and TMview.

Word Mark
DevScan

Word Mark

The name DevScan set in JetBrains Mono, weight 600. The split between Dev (white) and Scan (electric blue #3B82F6) visually encodes the product's core proposition: developer-first security scanning.

Figurative Mark

Figurative Mark

An abstract shield (protection) containing a horizontal scan line with a pulse dot and a green checkmark (successful analysis). Used as an app icon and favicon without accompanying text. Designed as original SVG artwork by NexScan GmbH.

Color Mark
#3B82F6
#0F172A
#10B981

Color Mark

The distinctive combination of Electric Blue #3B82F6 on Dark Navy #0F172A, accented with Emerald #10B981. The palette communicates technical precision and trust — deliberately distinct from the reds and greens of established security brands.

Nice Classification

Our marks are registered under the following Nice classes. We verified no conflicting existing marks using EUIPO eSearch and TMview prior to registration.

Class Category Goods & Services covered
09 Software & Technology Computer software; security analysis software; downloadable software applications; software for detecting vulnerabilities in computer code; software for dependency scanning and static code analysis.
42 IT & SaaS Services Software as a Service (SaaS); cloud-based software services; IT security consultancy; providing temporary use of non-downloadable software for code analysis and security scanning.

Conflict check note: We deliberately avoided red/green shield combinations common in established security brands, names phonetically similar to Snyk, Veracode, or Checkmarx, and blue hexagonal logos similar to Docker. A search on EUIPO eSearch for "DevScan" in classes 9 and 42 returned no conflicting marks as of May 2025.
